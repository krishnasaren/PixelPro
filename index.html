The Core Feature
Edit Every Single Pixel

No other editor gives you this level of control. Zoom in until individual pixels are visible as large squares, then click to paint, inspect, or batch-edit. Every pixel exposes its raw RGBA data.

  • ✓ Click any pixel to view its exact R, G, B, A values
  • ✓ Shift+click to select multiple pixels simultaneously
  • ✓ Batch edit — change all selected pixels at once
  • ✓ Real-time pixel inspector with hex and decimal values
  • ✓ Select all pixels matching a specific color
  • ✓ Drag to paint across multiple pixels
Code-Driven Editing
Write Code,
Transform Images

Access the raw pixel array of any layer and transform it programmatically. The built-in code editor gives you a full JavaScript and Python API to manipulate images with precision that no GUI tool can match.

  • → Loop through every pixel and apply custom mathematical transformations
  • → Write custom filters beyond what the UI offers
  • → Use Python with NumPy and Pillow for server-side execution
